## 📖 About The Project

As a BSc.CS student passionate about building intelligent systems, I designed this portfolio to be more than just a list of links—it's an interactive experience. It features modern web design principles, dynamic animations, mesh gradients, and a custom light/dark theme transition. 

### Key Features
*   **Modern UI/UX:** Glassmorphism, animated backgrounds (auroras, particles, stars), and smooth scrolling.
*   **Responsive Design:** Fully optimized for both desktop and mobile devices.
*   **Interactive Elements:** Custom cursor, theme toggling, and animated project cards.
*   **Performance:** Lightweight vanilla JavaScript and CSS without heavy frameworks.

## 🛠️ Built With

*   **HTML5** - Semantic structure and SEO optimization.
*   **CSS3** - Vanilla CSS for styling, custom animations, and responsive flexbox/grid layouts.
*   **JavaScript (Vanilla)** - DOM manipulation, scroll events, and interactive logic.
*   **FontAwesome & Devicon** - For professional scalable icons.
*   **AOS (Animate On Scroll)** - For scroll-based entrance animations.
